Can you think of an analogy to describe HTML, CSS and the DOM?

I would use an analogy of a microwave oven. HTML would be the technolegy inside the microwave that figures out how long to cook something, the temperatue, what to do when it's done and so on. CSS would then be the way the microwave looks; its colour, size, weight etc. DOM is the person using the microwave. They decide overall what they are cooking, the settings needed to cook, to stop the cook early and more.

What is meant by boxifying design?

This is when you seperate each component of a web page design into boxes. For example putting the body of a page into a box and splitiing each paragraph into another individual box. Then doing the same to the images, links, titles etc. until every last component has it's own box.

What is the box model?

Putting more boxes around your boxes. The original box in the middle is called 'content'. Around that you have margins, borders and padding.
