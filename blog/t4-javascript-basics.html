HTML/CSS and JavaScript

Each of these all do different things to have the outcome of a well structured, visually pleasing and interactive website. HTML is used to add all the information of the website. Eg. The text, headings, nav bar. CSS is used to make all the elements in the HTML look a certain way. Eg. Image size, font, background. JavaScript is used to create the interactive aspect of a website. Eg. Inputing information to be used by the website.

Control flow and loops

Control flow is a very important part of programming because it allows the code to do different things depending on the scenario. This is done in many different ways. If/else, switch and loops are three way in which this is done. For example lets say I’m tired and I’m asking a website what to do (I hope most people are able to figure this out without the use of a website). Here we are using if/else statements to tell the user what to do: 

            
              var userAnswer = prompt("Are you tired?");

              if (userAnswer === "yes") {
                console.log("Go to sleep!")
              }
              else {
                console.log("Stay awake!")
              }
But what if we want to have multiple different answers? We could use "if else" statements but there is a way more efficent way of doing it. Here we are using switch statements to tell the user what to do: 
            
              var userAnswer = prompt("Are you tired?");

              switch (userAnswer) {
                case "yes":
                  console.log("Go to sleep!")
                  break;
                case "no":
                  console.log("Don't sleep!")
                  break;
                case "a bit":
                  console.log("Have a quick nap!")
                  break;
                case "i don't know":
                  console.log("Figure it out and come
                  back later")
                  break;
                default:
                  console.log("I didn't understand that.
                  Try again!")
              }
How about if we wanted to ask the user multiple times? We would use a loop. Here we are using a for loop and we are asking 5 times. 
            
              var userAnswer = prompt("Are you tired?");
              for(i=0; i < 5; i++) {
                i++;
                switch (userAnswer) {
                  case "yes":
                    console.log("Go to sleep!")
                    break;
                  case "no":
                    console.log("Don't sleep!")
                    break;
                  case "a bit":
                    console.log("Have a quick nap!")
                    break;
                  case "i don't know":
                    console.log("Figure it out and come
                    back later")
                    break;
                  default:
                    console.log("I didn't understand that.
                    Try again!")
                }
              }
These are all examples of control flow statemets.
