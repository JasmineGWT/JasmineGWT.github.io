parentheses ()

There are two uses for parentheses in JS. The first one is the order of operations in mathmatical statements eg. 7*(4+1) = 35 . It is used the same way as it would be in any mathmatical statement.

The other use is in functions. They can either call functions or pass parameters and arguments into functions. Calling functions looks like this:

prettyDopeFunction()

Passing parameters and arguments looks like this:

prettyDopeFunction(argument)

brackets []

Square brackets can be used to either make an array or acces information within an array. To make an array it would look like this:

var prettyDopeArray["Hello", 6, true]

Each item in an array is assigned a number in terms of which items which. These numbers start at 0. So in our array above, "6" would be item 1. To acces it would look like this:

prettyDopeArray[1]

braces {}

Braces are used to enclose a block of code. They are used in functions, but also if statements, while and for loops. For example:

var prettyDopeFunction() { ##DO THIS THING I WANT YOU TO DO## }

if (prettyDopeExample > 7) { ##DO THIS THING I WANT YOU TO DO## }

single quotes '' and double quotes ""

Both of them are the same thing in JavaScript, just a personal preferance thing. They are used to enclose a string. That would look like this:

var lastDopeExample = "I am not funny"
