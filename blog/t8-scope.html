What is scope?

Scope is what we have acces to and where. There are two types of scope, global and local.

Global is when a variable is declared outsite of any function, meaning that it can be accessed from anywhere. Local is when a varible is declared within a function and can only be accessed within that function.

A way to look at it would be if you save a file somewhere on the internet that you can access it with any devise, this would be global. However if you save it only on your desktop it would only be accessable through your desktop, therefore it would be local.
