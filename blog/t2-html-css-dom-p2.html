What is grid based design?

This is when you split a web page into a grid and put different elemets of your web page into different parts of your grid. This framework consists of rows and columns.

Whats all the hype about responsive webpages?

A responsive page is one that changes acording to the device in which its being viewed on. For example, if you view this website on a laptop it will look slightly different to if viewed on a phone. The reason this is 'hyped' is because today with so many different types of devices we need to be sure that no matter how you view a webpage, it looks the way its intended to look.

What is semantic structure?

Semantic structure is the when we layout our HTML in a way that both the browser and a human could can read and understand the code. We do this using tags such as h1, h2, div, body and many more.
